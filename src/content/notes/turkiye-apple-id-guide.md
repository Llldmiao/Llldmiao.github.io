## 2. 如何通过 Gmail 注册土耳其区 Apple ID

一种成功率较高的路线是：先注册中国大陆区 Apple ID，再通过网页端把账号地区改为土耳其区。

![土耳其区 Apple ID 注册流程图](/images/turkiye-apple-id-register-guide.jpg)

### 重要提示

- 全程使用全局模式稳定网络（美国/日本节点即可）
- 转区后建议立即下载 1-2 个免费 App 养号
- 如遇 None 选项消失：检查余额为 0、无未完成订阅、换浏览器或网络重试

### 操作步骤

1. 注册中国大陆区 Apple ID
   - 使用浏览器无痕模式打开 https://account.apple.com/
   - 选择创建 Apple ID。
   - 国家或地区先选择中国大陆。
   - 使用干净邮箱（未注册过任何 Apple ID 的 163/QQ/Gmail）和 +86 手机号填写
   - 出生日期建议填写为 18 岁以上。
   - 完成邮箱验证和短信验证。
   - 注册完成后，先不要在手机 App Store 里登录这个账号。
2. 网页端转区到土耳其（最关键一步）
   - 打开土耳其区 Apple Account 入口：https://account.apple.com/tr/
   - 登录刚注册的中国大陆区 Apple ID。
   - 进入「个人信息」→「国家/地区」→ 点击「更改」
   - 同意条款后，在付款方式页面优先选择 None（无）
   - 账单地址填写土耳其地址，可使用地址生成器：https://appark.ai/cn/address-generator/tr-address（优先选伊斯坦布尔地址）
   - 电话继续保留你的 +86 手机号
   - 点击保存，确认账户国家变为 Türkiye
3. 手机端切换 App Store 到土耳其区
   - iPhone/iPad 设置 → 顶部点击你的 Apple ID →「媒体与购买项目」→ 退出登录（必须退出）
   - 用 Safari 打开以下 App Store 土耳其区跳转链接（出现“无法连接 App Store”属于正常）：

     ```text
     itms-apps://itunes.apple.com/WebObjects/MZStore.woa/wa/resetAndRedirect?dsf=143480&cc=tr
     ```

   - 返回 App Store，使用已经转区的 Apple ID 重新登录。
   - 检查账户页面，确认地区显示为土耳其。
4. 下载免费 App 固定地区（必须操作，否则容易回滚）
   - 在土耳其区 App Store 搜索任意免费应用（例如“免费游戏”或“WhatsApp”）
   - 点击“获取” → 下载
   - 如果下载过程中再次要求填写地址，继续填写土耳其地址，付款方式选择 None（无）
   - 下载成功后，地区彻底固定，后续订阅、充值都不会轻易变回国区
